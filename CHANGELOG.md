# Changelog

Alle wichtigen Aenderungen an der DocFlow Scanner Bridge werden hier dokumentiert.

Format basiert auf [Keep a Changelog](https://keepachangelog.com/de/1.0.0/).

## [1.0.0] - 2026-02-13

### Hinzugefuegt
- Automatische Scanner-Erkennung (mDNS/Zeroconf, WSD, IP-Scan)
- QR-Code und manuelles Pairing mit DocFlow
- System Tray Integration (Hintergrund-Betrieb)
- eSCL/AirPrint Scanner-Zugriff
- Scan-Job-Polling und automatischer Upload
- Sichere API-Key-Speicherung im OS-Keyring
- Auto-Updater via GitHub Releases
- Cross-Platform: Windows, macOS (Intel + ARM), Linux
